# CPU Fan Patrol

Small static web game for a 3-year-old who likes CPU fans.

## What it does

- Click rounds highlight one fan by color with a floating hand pointer.
- Typing rounds show a single large letter to press.
- Square SVG fans are styled to look more like PC case / CPU fans.
- A rainbow center fan stays as the special fan.
- Simple synthesized sound effects can be toggled on or off.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://127.0.0.1:4173`.
